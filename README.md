# ***Цель исследования:***
Разработать аналитический дашборд для компании организатора TED конференций, который позволит: выявлять наиболее востребованные темы и оценивать реакцию аудитории на них; определять оптимальные параметры мероприятия (число выступающих, длительность выступлений, хронометраж конференции); выделять наиболее привлекательных для аудитории спикеров и понимать, чем они интересны; опираться на реальные данные прошедших конференций для планирования первого мероприятия в соответствии с канонами TED.
# ***Результат:***
1.	Определена структура дашборда исходя из ключевых вопросов:
- общий обзор данных (объём и надёжность выборки);
- аналитика по конференциям (с детализацией по странам);
-	анализ выступлений (популярность, юмор, теги);
- профили выступающих (род деятельности и вовлечённость аудитории);
- детальный просмотр отдельных выступлений.
2.	Сформулированы ключевые метрики для каждого раздела:общий обзор, Анализ конференций, Анализ выступлений, Анализ выступающих.
3.	Продумана интерактивность:добавлены селекторы , обеспечена детализация от общего обзора до конкретного выступления.
